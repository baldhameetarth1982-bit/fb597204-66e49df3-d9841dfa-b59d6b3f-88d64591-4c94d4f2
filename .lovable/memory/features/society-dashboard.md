First screen shows (in order of importance):
- Pending Maintenance (count + amount)
- Outstanding Collections
- Cash Balance (derived)
- Bank Balance (derived)
- Total Balance
- Visitors Today
- Pending Resident Approvals
- Recent Income
- Recent Expenses
- Complaints (open)
- Recent Activity
Color signals: green growth / red loss / white neutral on the finance chart.
