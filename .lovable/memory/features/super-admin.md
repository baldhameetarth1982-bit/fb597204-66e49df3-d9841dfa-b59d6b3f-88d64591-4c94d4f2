description: Platform-level features for super admin only

---
Super Admin owns ONLY platform tools. No society/resident management mixed in.
Features (build progressively):
- Society Management (list, search, suspend, force-cancel)
- User Management (search any user, grant plan/discount/lifetime)
- Subscription Plans CRUD (Basic, Premium, Master, Custom)
- Custom Plan Pricing & per-Module Pricing
- Discount Management
- Lifetime Access grants
- Revenue Dashboard & Payment Analytics
- Advertisement Management (max 4 ads, placement + duration 10–30s)
- Razorpay attach + verification
Never reuse desktop sidebar from society admin — Super Admin is its own SaaS dashboard.
