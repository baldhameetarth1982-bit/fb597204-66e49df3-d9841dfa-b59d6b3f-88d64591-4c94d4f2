type: feature

**Guard Links** (Phase 6): admin generates Permanent or Temporary links. One active device per link. Second-device attempt notifies admin; admin can Revoke Session, Reset Device, Generate New Link.
**Guard interface**: only 4 actions — Visitor Entry, Visitor Exit, Today's Visitors, Search Visitor. Plus a Manual Walk-In for unexpected visitors.
**Residents**: create Expected Visitors and Frequent Visitors (maid, cook, driver, tutor, milkman). Validity: Today, Tomorrow, Custom Date, Recurring.
QR-code-first flows are out. Keep it simple.
